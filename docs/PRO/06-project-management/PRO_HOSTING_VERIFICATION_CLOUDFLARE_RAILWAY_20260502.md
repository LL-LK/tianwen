## 一、执行摘要

本报告对天问-AGI项目计划使用的 Cloudflare Pages（前端静态托管）和 Railway（后端Docker部署）两个平台的免费/低成本计划进行全面的可靠性验证。结论：**Cloudflare Pages 完全可靠且免费，Railway Hobby计划可满足初期需求但需注意资源限制**。

### 4.1 前端部署 (Cloudflare Pages)

1. 连接 GitHub 仓库
2. 构建命令: 无需（纯静态）
3. 输出目录: web/
4. 环境变量: 无需
5. 自定义域名: 绑定观测站域名

### 4.2 后端部署 (Railway)

1. 连接 GitHub 仓库
2. 自动检测 Dockerfile
3. 设置环境变量:
   - `PORT=5000`
   - `DEBUG=false`
   - `API_KEY=<生成强密钥>`
   - `CORS_ORIGINS=https://your-domain.pages.dev`
4. 健康检查路径: `/api/health`
5. 卷挂载: `/app/data` 用于持久化

## 六、最终结论

### Cloudflare Pages: ✅ 强烈推荐
- 免费、无限流量、全球CDN、商业许可
- 前端静态托管的最佳选择
- 2025年维护模式不影响使用

### Railway Hobby: ⚠️ 可用但需监控
- $5/月极低成本，Docker原生支持
- 0.5GB RAM对ML工作负载偏紧
- 建议：初期使用Hobby计划，若OOM频繁则升级Pro($20/月)
- 备选：若Railway不稳定，可快速迁移到Fly.io或自建VPS

### 总体评估: ✅ 托管计划可靠
Cloudflare Pages + Railway Hobby 组合可以满足天问-AGI的初期部署需求，总成本$0-5/月，可靠性良好。建议在Railway上配置内存监控告警，并准备Pro计划升级路径。
