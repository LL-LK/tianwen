## 二、AstroIR纠错确认

### 2.1 错误详情

| 属性 | Issue原文 | 实际情况 |
|-----|---------|---------|
| **项目名称** | AstroIR | AstroIR (正确) |
| **项目类型** | 天文基础模型 (Foundation Model) | **数据集/基准测试 (Dataset/Benchmark)** |
| **论文编号** | - | arXiv:2306.03138 |
| **发布时间** | - | 2023年 |
| **开发团队** | - | Ziyang-Li-AILab |

### 2.2 正确描述

AstroIR是正儿八经的天文感知基准测试数据集，而非基础模型：
- **论文**: "AstroIR: Toward Benchmarking Spatial, Spectral, and Temporal Patterns of Astronomical Images"
- **用途**: 评估天文图像处理算法在空间、光谱、时间维度上的综合表现
- **数据规模**: 包含模拟和真实天文图像，覆盖多种观测场景
- **与天问集成**: 可作为测试基准，但不适合作为"模型能力"集成

### 2.3 纠错建议

Issue #4应明确标注:
```
❌ AstroIR 不是天文基础模型
✅ AstroIR 是天文图像基准测试数据集
```
