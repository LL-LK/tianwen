## Key Observations
1. Multiple astronomy AI projects exist but few combine true AGI capabilities with astronomical data
2. Galaxy classification using CNNs is well-established (since 2016)
3. Exoplanet detection using ML on Kepler/TESS data is active
4. Telescope scheduling optimization is a niche but active area
5. Langchain-based AI agents for astronomy are emerging
6. Real-time 3D universe exploration is a new direction (CosmoPilot)
7. No strong evidence of projects combining astronomical data processing with general AI reasoning capabilities
