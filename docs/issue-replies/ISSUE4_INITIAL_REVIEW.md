### 2.1 AstroIR 事实性错误 - ⚠️ 必须更正

| 属性 | Issue原文 | 实际情况 |
|-----|---------|---------|
| **项目名称** | AstroIR | AstroIR (正确) |
| **项目类型** | 天文基础模型 (Foundation Model) | ❌ **错误** |
| **真实类型** | - | **数据集/基准测试 (Dataset/Benchmark)** |
| **论文编号** | - | arXiv:2306.03138 |
| **发布时间** | - | 2023年 |
| **开发团队** | - | Ziyang-Li-AILab |
| **定位** | 天文基础模型 | 天文感知基准测试数据集 |

**AstroIR 正确描述**:

```
AstroIR 是天文学领域的数据集/基准测试项目，而非基础模型。
- 论文: "AstroIR: Toward Benchmarking Spatial, Spectral, and Temporal Patterns
        of Astronomical Images" (arXiv:2306.03138)
- 用途: 评估天文图像处理算法在空间、光谱、时间维度上的表现
- 包含: 模拟和真实的天文图像数据
```

**影响分析**:
- 如果Issue #4将AstroIR列为"天文基础模型"，这会误导后续技术选型
- 应改为"天文图像基准测试数据集"
- 天问-AGI 应寻找真正的天文基础模型进行集成
