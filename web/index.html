天问-AGI 全自动天文观测站

天问-AGI 是一个全自动天文观测系统，支持无人值守自主发现与观测。以下为系统功能说明：

1. 观测总控台

显示观测站实时状态，包括当前观测目标、设备状态、观测队列等。连接真实望远镜后，此处将显示实时数据。

2. 实时星图

通过 NASA SkyView API 获取真实天文图像。支持 DSS、2MASS、SDSS 等多种巡天数据。输入天体名称（如 M31、M42）即可查看。

3. 数据面板

展示观测数据统计，包括检测结果、光变曲线、研究周期历史等。

4. 研究闭环

自动研究循环：文献检索 → 假说生成 → 假说检验 → 发现确认 → 观测调度 → 自我进化。

5. 告警中心

显示系统告警信息，包括新天体发现、天气变化、设备异常等。

6. 望远镜控制

控制望远镜进行 GOTO 指向、跟踪、曝光等操作。需要连接真实望远镜设备后方可使用。

7. 智能对话

与 AI 助手对话，查询天体信息、获取操作帮助。需配置 API Key 以启用 MiniMax LLM 功能。

8. 系统日志

查看系统运行日志，按级别筛选（发现、检测、望远镜、相机、系统）。

️ 关于数据真实性

当前系统未连接真实望远镜，观测总控台显示的是空状态。连接真实设备后，所有数据将来自实际观测。

实时星图数据来自 NASA SkyView API，是真实的天文图像。

天体坐标数据来自内置星表，基于真实天文数据。

快捷键: 1-6 切换面板 · R 刷新数据 · T 连通性检测
